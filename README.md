# ONE: Challenge Telecom X - Parte 2 (Machine Learning)

Esta análise foi feita como solução de um desafio do programa [ONE - Oracle Next Education](https://www.oracle.com/br/education/oracle-next-education/). Trata-se de uma continuação direta do projeto [Telecom X](https://github.com/leonardo-queiroz/challenge-telecom-x), onde foram analisados os dados de uma empresa fictícia, com o objetivo de identificar se ela está sofrendo evasão de clientes.

Esse projeto se aprofunda ainda mais na análise de churn. Usando as bibliotecas Matplotlib e Seaborn para criar uma matriz de correlação das variáveis numéricas presentes no banco de dados. E usando a biblioteca Scikit-learn para a criação de dois modelos preditivos, um de Regressão Logística e outro de Random Forest.

Para acessar o arquivo notebook (.ipynb) e poder ver o código, os gráficos, bem como as tabelas mostrando o desempenho desses modelos e o relatório, basta clicar no arquivo "telecom_x_parte_2.ipynb" presente no repositório e você conseguirá acessá-lo pela própria plataforma do GitHub.
